                 Source code ---> Human made language [English]
                 Byte Code ----> Machine Language [Binary Digits --> 0 1]

Java Architecture:-
*******************
1.JVM :- Java Virtual Machine
         ->Physically not exists
         ->it is used for memory allocation

2.JRE :- Java Runtime Environment
        ->it help to run the program
        ->Predefined files(.class files)

3.JDK :- Java Development Kit
         ->JDK = JVM + JRE
         ->JDK version 1.0 - 1.16 ,1.17
         ->SE 1.8 - Standard Edition


Coding standards:-
******************
1.Pascal:-
         ->Every words first letter must be in capital
      Eg:-
         -Selenium_TestingToolAutomation //Demo_Test
         ->Project/Class Name

2.Camel:-
         ->Except first word remaining words first letter must be capital.
      Eg:-
         ->selenium _TestingToolAutomation   //demo_Test
         ->Method/Variables/Object Name
